# Hardware:

LCD - https://www.waveshare.com/7inch-hdmi-lcd-h.htm  ->  For this any size that you want is perfectly fine, it is recommeded to get as high of a peak-nit brightness as possible though

Retroreflective Material - https://americansignproducts.com/product/3m-3930-white-high-intensity-prismatic-reflective-sheeting/ -> For the US this was the best we could find, ideally you would want a smooth material as this tape has lines and causes a bluriness to images.

Beam Splitter - https://glidegear.net/products/teleprompter-replacement-glass -> It appears that the best option for small beam splitter glass is teleprompter replacement glass. 

When choosing all three of the above hardware it is important that the size matches the LCD as you want to reflect all of the LCD screen, so choosing a retroreflector and beamsplitter larger than the LCD is advised
